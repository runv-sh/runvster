## Current Status

Implemented today:

- Docker-first local stack
- account creation
- login via `/login`
- server-side sessions
- user profiles
- link posts
- text posts
- tags
- feed filters: `Recentes`, `Top`, `Links`, `Discussao`

Still missing:

- threaded comments
- votes
- ranking based on votes/activity
- moderation tools
- admin panel
- production deploy artifacts
