## Direction

- bold red accents
- high contrast
- dense reading surfaces
- minimal chrome
- feed-first layout

## UI Rules

- prefer list rhythm over large marketing blocks
- avoid generic SaaS landing aesthetics
- use red mainly for active state, links, and important actions
- keep typography readable and compact
